## ü§î About The Project

A great way to reward your students for completing the course is to issue a certificate of completion. Easily customize your certificates.

Beautiful certificate your students are proud to display.

**-** <a href="https://frontend-custom-certificates.herokuapp.com" target="_blank">Frontend</a>

<!-- INSTALLATION -->

## üî® Installation Default

You'll need [Node.js](https://nodejs.org) installed on your computer in order to build this app.

```bash
git clone https://github.com/lucioerlan/Custom_Certificates.git
$ cd Custom_Certificates
$ npm install
```
